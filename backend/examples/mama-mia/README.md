# MAMA-MIA Pre-loaded Example

This directory contains the pre-loaded MAMA-MIA example for demonstrating VeriFlow capabilities.

## Contents

- `context.txt` - Supplemental context for the Scholar Agent
- `ground_truth_isa.json` - Expected ISA-JSON output for validation
- `sample_pdf.txt` - Placeholder for PDF (see instructions below)

## Adding the Real PDF

To use an actual MAMA-MIA paper:

1. Download the MAMA-MIA paper PDF
2. Rename it to `paper.pdf` and place it in this directory
3. The system will automatically detect and use it for the demo

## Usage

The pre-loaded example is automatically available when starting VeriFlow.
Select "MAMA-MIA Demo" from the upload panel to load the example.
